# Tik-Tok-Clone-Backend
### This Backend was developed using Express, Node and Mongo DB.

It provides REST Endpoints to acess tik-tok-clone Video contents

> try api: https://tik--tok--clone--backend.herokuapp.com

> hit endpoint: /posts

```
collection 'posts':
    url: String,
    channel: String,
    desc: String,
    song: String,
    likes: Number,
    shares: Number,
    messages: Number
```
